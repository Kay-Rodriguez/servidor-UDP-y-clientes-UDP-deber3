# Deber 3 - UDP Cliente Servidor con JavaFX

##  Descripción

Este proyecto implementa una aplicación cliente-servidor utilizando el protocolo UDP en Java.
El sistema permite enviar preguntas desde un servidor y responderlas desde un cliente con interfaz gráfica desarrollada en JavaFX.
El objetivo es demostrar la comunicación mediante datagramas, así como la validación de respuestas en tiempo real.

---

##  Funcionamiento

### 🔹 Servidor

* Escucha en el puerto 5000
* Envía preguntas sobre desarrollo de software
* Valida las respuestas del cliente

### 🔹 Cliente

* Interfaz gráfica en JavaFX
* Permite obtener preguntas
* Permite enviar respuestas
* Muestra si la respuesta es correcta o incorrecta

---

##  Flujo del sistema

1. El cliente solicita una pregunta
2. El servidor envía la pregunta
3. El usuario ingresa una respuesta
4. El cliente envía la respuesta al servidor
5. El servidor valida la respuesta
6. El cliente muestra el resultado
